This page summarizes my worklog throughout the **12-week** internship at Amazon Web Services Vietnam (from April to July 2026). It reflects a journey that began with foundational cloud computing concepts and progressed toward designing and building a complete, real-world system on AWS.

The internship was organized into two phases:

**Phase 1 – Building an AWS Foundation (Week 1 – Week 7):** Explored AWS's core services, including EC2, S3, VPC, RDS, IAM, CloudFront, Route 53, SageMaker, Lambda, and EventBridge, along with key security practices, while preparing for the AWS certification exam.

**Phase 2 – Delivering the FlashLearn Project (Week 8 – Week 12):** Built FlashLearn, an English-learning application on AWS, featuring flashcards, quizzes, battles, a community space, a leaderboard system, and final project handover.

**Week 1:** [Getting started with AWS, setting up an account, and understanding cloud computing basics](1.1-week1/)

**Week 2:** [Exploring the AWS service ecosystem further, hands-on practice with EC2 and EBS](1.2-week2/)

**Week 3:** [Studying EC2 architecture, load balancing with ELB, Auto Scaling, containers, serverless computing, and database services](1.3-week3/)

**Week 4:** [Performing data migration, configuring VPC networking, managing DNS with Route 53, and content delivery via CloudFront](1.4-week4/)

**Week 5:** [Storage solutions with S3/EBS/EFS, applying AI/ML with SageMaker, data analysis, and service integration](1.5-week5/)

**Week 6:** [Strengthening AWS security knowledge with the Shared Responsibility model, GuardDuty, and advanced IAM](1.6-week6/)

**Week 7:** [Consolidating everything learned so far and preparing for the AWS certification exam](1.7-week7/)

**Week 8:** [Applying Infrastructure as Code with AWS CDK and Terraform, and getting introduced to NodeJS](1.8-week8/)

**Week 9:** [Kicking off the FlashLearn project with a multi-AZ VPC architecture, ALB, S3/CloudFront, and Lambda combined with Polly](1.9-week9/)

**Week 10:** [Improving system performance, automating image processing, API testing, and setting up monitoring](1.10-week10/)

**Week 11:** [Designing core APIs, building the content approval workflow, using S3 presigned URLs, and data modeling with an ERD](1.11-week11/)

**Week 12:** [Developing community features, a ranking system, nested comments, and finalizing the project handover](1.12-week12/)
